# Electron Frontend

Desktop UI for ccBenchmarkTool built with Electron, React, and TypeScript.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
# Development
npm run dev

# Production build
npm run build
```

## Tech Stack

- Electron 35
- React 19
- TypeScript
- Vite
- Zustand (state management)
- Tailwind CSS
- shadcn/ui components
